Market Analysis
The Five Biggest Unsolved Gaps
Where every incumbent fails — and where Astra wins. These are structural gaps, not temporary feature misses.
01
Business-logic / authorization vulnerabilities at PR time
49% of high/critical bug-bounty findings live here (Semgrep, Nov 2025). IDOR, BOLA, BFLA, broken access control — invisible to traditional SAST. ZeroPath: 53% of their critical disclosures are authorization flaws. Semgrep's AI beta is private and IDOR-only. Pure-LLM approaches run 23–65% precision (NDSS 2025) and 78% false-positive rate (Semgrep Claude Code study). No incumbent has shipped a production-grade hybrid engine for this at PR time.
→ Astra: repo_intel + cross_file graph + hybrid deterministic+LLM. Business-logic as lead marketing message and public benchmark proof point.
02
A unified transparent-priced platform: AI PR review + AI SAST + reachability SCA + malicious-package detection + agentic auto-fix
Every security team buys 4–6 tools today. CodeAnt AI is the closest, but SAST depth is unproven on logic flaws and market presence is minimal. CodeRabbit has review but no deep SAST. Snyk has SCA but no conversational PR review. No single product delivers all five capabilities with transparent self-serve pricing.
→ Astra: single platform, single dashboard, single price. Target the buyer who has CodeRabbit + Snyk + Gitleaks + Checkov running as four separate tools.
03
Multi-repo / cross-service / monorepo reasoning
CodeRabbit just launched a limited multi-repo feature in March 2026, confirming this is the next battleground. No incumbent has solved cross-service data flow analysis. Auth defined in service A, bypassed in service B — invisible to all current PR reviewers. Microservices architectures have entirely undetected cross-service IDOR vulnerabilities.
→ Astra: repo_intel builds SystemIntelReport per service; cross_file clusters by domain. Phase 3 roadmap: cross-repo graph and service-mesh analysis.
04
AI-aware SAST tuned for AI-generated (vibe-coded) patterns
SafeGenBench (arXiv Jun 2025): 558 test cases, 8 languages — "even advanced LLMs yield vulnerable code." AI-generated code has distinct vulnerability patterns vs human-written code. Endor Labs AURI, Sonar AI Code Assurance, Semgrep multimodal beta all targeting this. The "different model writes vs reviews" principle is the emerging architecture.
→ Astra: tag AI-generated files during repo_intel; apply AI-code-specific prompt overlays in deep_scan. Use a different model family to review than typically generates the code.
05
Pricing accessibility for the 11–49 developer band
Snyk/Semgrep/Veracode all force enterprise sales motions above 10 devs. The 11-dev team hits ~$105/dev/month at Snyk's enterprise tier. CodeRabbit ($24) and Aikido ($350 flat) understood this gap before most security vendors. The 11–49 dev market is large, underserved, and has PLG purchase patterns — no sales call required.
→ Astra: Free OSS → $20 entry → $40 Pro → $80 Enterprise per active PR author. Transparent, self-serve, no sales call under $2K/month. PLG motion from free trial.

State of the Art
What SOTA means in 2026
10 mandatory capabilities. No single incumbent delivers all 10. Academic evidence converges on hybrid deterministic+LLM as the only architecture achieving both high recall and acceptable precision.
VULSOLVER (arXiv 2509.00882) — The Key Research Finding
Hybrid deterministic+LLM (Semgrep dataflow + DeepSeek-V3 reasoning): 96.29% accuracy, 96.55% F1, 100% recall on OWASP Benchmark (1,023 labeled samples). 15 previously unknown high-severity vulnerabilities discovered in real GitHub repos (CVSS 7.5–9.8). Ensembling SAST + LLM "mitigates the drawbacks of both" — SAST's low recall and LLM's high false positives. This is the architecture. Build it.
01 · CRITICAL
Hybrid deterministic + LLM detection engine
Semgrep-style AST/dataflow grounding + frontier model ensemble. 1.9× IDOR recall improvement (Semgrep beta). 78% false-positive rate with pure LLM alone — grounding is mandatory. VULSOLVER: 96.29% accuracy.
02 · CRITICAL
Whole-codebase code graph with reachability analysis
Not file-by-file. Greptile/Endor Labs-style. 95% of CVEs live in transitive dependencies — reachability is the only way to know if they're actually exploitable. Cross-file auth gaps are completely invisible without the full graph.
03 · CRITICAL
Business-logic vulnerability detection (IDOR/BOLA/BFLA)
49% of critical bug-bounty findings. Near-zero traditional SAST detection rate. Requires system-level context from repo_intel + cross-file reasoning. The single largest unserved gap — and Astra's primary wedge into the market.
04 · HIGH
Multi-Git-platform: GitHub + GitLab + Bitbucket + Azure DevOps
Only CodeRabbit and CodeAnt AI span all four. Greptile is GitHub/GitLab only. GHAS is GitHub only. Platform breadth is a hard moat — enterprise procurement requires it. No workarounds.
05 · HIGH
MCP / agent-friendly architecture for AI coding agent governance
Endor Labs AURI: "verifiable evidence — data flow, call paths, reachability — for every finding." Mend.io shipped first SAST MCP server. Claude Code, Cursor, Copilot, Codex, Devin all need a policy/verification layer. This is the forward moat.
06 · HIGH
Agentic auto-fix that opens human-approved PRs
Pixee: "humans approve every change." Never auto-merge. CodeRabbit Autofix + Cursor BugBot Autofix are the proof points. The 74% manual triage time reduction claim (Pixee) is the commercial hook for enterprise buyers who are drowning in findings.
07 · HIGH
Memory-based learning from triage decisions
Semgrep Assistant Memories: ~20% triage automation baseline that rises with org context. Cursor BugBot "Learned Rules." Every customer's triage history makes Astra smarter for that customer — the long-term data moat incumbents cannot replicate retroactively.
08 · HIGH
Self-hosted / air-gapped / VPC deployment
Commvault (800+ engineers) on-prem. Bajaj Finserv Health on Azure DevOps. Qodo self-hosted, CodeAnt air-gapped. Regulated industries (finance, healthcare, defence, government) require it by policy. No enterprise deal closes without it.
09 · MEDIUM
Transparent self-serve pricing in the $20–$50/dev band
The 11–49 dev gap where Snyk/Semgrep/Veracode all force enterprise sales. CodeRabbit ($24) and Aikido ($350 flat) understood this first. PLG motion drives the top-of-funnel from which enterprise deals eventually close.
10 · MEDIUM
Supply-chain: malicious packages + SBOM + pinned Actions
Shai-Hulud npm worm (Sep 2025): 500+ packages compromised. Trivy-action poisoned twice Mar 2026: 76/77 release tags. Socket blocks 100+ attacks/week. Pin Actions to commit SHAs. Malicious-package detection is table-stakes in 2026 — no longer a differentiator.
